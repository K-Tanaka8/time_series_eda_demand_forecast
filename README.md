## 概要

このプロジェクトは、需要予測（売上予測）における時系列データの探索的データ分析（EDA）を示すサンプルです。
Pythonを用いて売上データのトレンド、季節性、曜日別傾向、月別傾向、自己相関などを視覚的に確認できます。

## データについて

このNotebookでは以下の特徴を持つダミー売上データを生成しています。

売上の長期トレンド  
週次の季節性  
ランダムノイズ  

```python
sales = (
    50
    + 0.2 * np.arange(180)
    + 10 * np.sin(np.arange(180)/7)
    + np.random.normal(0, 3, 180)
)
```

実際の需要予測データを想定した構造になっています。

## 分析内容

### 1 基本統計量

売上データの平均、標準偏差、最大値などを確認します。

```python
df.describe()
```

### 2 時系列プロット

売上の時系列推移を可視化します。  
長期トレンド、季節性、ノイズの存在を確認できます。

### 3 移動平均

7日移動平均を計算し、売上トレンドを滑らかに表示します。

```python
df["rolling_mean_7"] = df["sales"].rolling(7).mean()
```

移動平均はノイズの除去に役立ちます。  
トレンド確認にも役立ちます。

### 4 曜日分析

曜日ごとの平均売上を確認します。

```python
df["weekday"] = df.index.dayofweek
```

EC、小売、飲食などのビジネスでは曜日効果がよく見られます。  
曜日による売上変動を確認できます。

### 5 月別売上分析

月ごとの平均売上を可視化します。

```python
df["month"] = df.index.month
```

月次の季節性と年間トレンドを確認できます。

### 6 ラグプロット（Lag Plot）

ラグ特徴量を作成し、自己相関を確認します。

```python
df["lag1"] = df["sales"].shift(1)
```

売上に自己相関性があるので、時系列モデルの適用可能性が確認できます。

### 7. EDAまとめ

本EDAから以下の特徴が確認された。

1 トレンド
売上は時間とともに増加している。

2 季節性
約7日周期の季節性が存在する。

3 曜日効果
曜日による売上差が存在する。

4 自己相関
前日の売上が翌日の売上に影響する。
