Pythonを用いて、売上データの以下の特徴を可視化します。

* トレンド
* 季節性
* 曜日別傾向
* 月別傾向
* 自己相関

## データについて

このNotebookでは以下の特徴を持つ **ダミー売上データ** を生成しています。

* 売上の長期トレンド
* 週次の季節性
* ランダムノイズ

```python
sales = (
    50
    + 0.2 * np.arange(180)                 # トレンド
    + 10 * np.sin(np.arange(180)/7)        # 週次季節性
    + np.random.normal(0, 3, 180)          # ノイズ
)
```

実際の需要予測データを想定した構造になっています。

## 分析内容

### 1 基本統計量

売上データの平均・標準偏差・最大値などを確認します。

```python
df.describe()
```

### 2 時系列プロット

売上の時系列推移を可視化します。

確認ポイント

* 長期トレンド
* 季節性
* ノイズ

### 3 移動平均

7日移動平均を計算し、売上トレンドを滑らかに表示します。

```python
df["rolling_mean_7"] = df["sales"].rolling(7).mean()
```

移動平均は以下の分析に役立ちます。

* ノイズの除去
* トレンド確認

### 4 曜日分析

曜日ごとの平均売上を確認します。

```python
df["weekday"] = df.index.dayofweek
```

EC・小売・飲食などのビジネスでは
**曜日による売上の変動（曜日効果）** がよく見られます。

### 5 月別売上分析

月ごとの平均売上を可視化します。

```python
df["month"] = df.index.month
```

これにより以下を確認できます。

* 月次の季節性
* 年間トレンド

### 6 ラグプロット（Lag Plot）

ラグ特徴量を作成し、自己相関を確認します。

```python
df["lag1"] = df["sales"].shift(1)
```

ラグプロットでは以下を確認できます。

* 売上の自己依存性
* 時系列モデルの適用可能性
